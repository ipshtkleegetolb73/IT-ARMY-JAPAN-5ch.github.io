スマホでMHDDoSを使おう！

MHDDoSってなんぞや？(前置き)

まずDDoS攻撃とは、超絶簡単に言いますと田代砲の上位互換です。DDoSツールには様々な種類がありMiraiやAnonDiscordなどあり様々です。

ここではウクライナIT軍でも使用が推奨されるDDoSツール、MHDDoSのインストール方法から使い方まで解説していこうと思います。

※Androidのみ解説となります。ご了承ください。

下準備

では早速TermuxというLinuxエミュレータをインストールします。

ここのURLからどうぞ Termux

下の画像のようにインストールが終わりましたら開くを選択してアプリを開いてください。

開くとかっこいい画面が出来ます。厨二心が揺さぶられますね。

Pythonとgitをインストール

それでは以下のコマンドを一行ずつ入力してEnterで実行しましょう。

pkg install root-repo
pkg update
pkg install python python3-pip git

※どうやら2022/5/1から公式のリポジトリが使えない状態なのでその対処方法の後に解説します。
